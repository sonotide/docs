---

При использовании высокоуровневой `playback_session` Sonotide предоставляет полностью интегрированный 10-полосный графический эквалайзер. Так как WASAPI в Windows не имеет таких функций "из коробки", мы вшили алгоритмы напрямую в пайплайн обработки Media Foundation, чтобы вам не пришлось с нуля писать DSP-процессор.

## 1. Базовое использование

Чтобы управлять эквалайзером, вызывайте методы напрямую у вашей сессии воспроизведения:

```cpp
// 1. Включаем обработку
session.set_equalizer_enabled(true);

// 2. Применяем готовый пресет
session.select_equalizer_preset(sonotide::equalizer_preset_id::jazz);

// 3. Немного снижаем общую громкость выхода, чтобы избежать перегруза
session.set_equalizer_output_gain(-1.5f);
```

## 2. Точечная настройка

Если встроенных пресетов (`rock`, `jazz`, `pop`, `spoken_podcast`) недостаточно, вы можете манипулировать любой из 10 полос вручную:

```cpp
// Слегка приподнимаем саб-бас
session.set_equalizer_band_gain(0, 3.5f); 
```

<Warning>
  Агрессивное поднятие полос выше `+0dB` неизбежно вызывает цифровой клиппинг (хруст). Если вы сильно выкручиваете определенные частоты вверх, обязательно компенсируйте это снижением общего `output_gain_db`.
</Warning>

## 3. Чтение состояния для интерфейса (UI)

Вы можете в любой момент вытащить всю нужную информацию для отрисовки ползунков в вашем приложении через `session.equalizer_state()`.

## Примечание к дизайну

Эквалайзер является частью фреймворка, потому что это критично важный функционал для любого медиаплеера. Однако, если вашему приложению нужна сложная обработка со своими компрессорами, ревербом и параметриком, вам следует игнорировать этот слой настройки и применять свою собственную DSP-логику к сырым байтам из обычного `render_stream`.
